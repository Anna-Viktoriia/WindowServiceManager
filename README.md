Опис проєкту

Програмне забезпечення призначене для управління службами операційної системи Windows. 
Програма дозволяє користувачу переглядати список доступних служб та виконувати базові операції керування ними, зокрема запуск, зупинку та перезапуск.
Розробка спрямована на спрощення взаємодії користувача з системними службами та надання зручного інструменту для їх контролю.

Основні можливості:
1. Отримання списку служб Windows

2. Відображення статусу служби (Running / Stopped)

3. Запуск служби

4. Зупинка служби

5. Перезапуск служби

6. Оновлення списку служб

7. Пошук служб за назвою
